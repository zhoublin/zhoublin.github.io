关于这个站点 About

这个小站托管在 GitHub Pages 上，用最简单的静态页面 + 一点点 JavaScript 和样式完成。

文章使用 Markdown 编写，支持代码高亮、公式（MathJax）、表格、图片和视频嵌入。

如何使用？How to use

  1. 在 posts/ 目录里新建 Markdown 文件。
  2. 在 posts/index.json 中登记文章信息。
  3. 提交并推送到 username.github.io 仓库。
